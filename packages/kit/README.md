# @czo/kit

Core toolkit for the c-zo modular e-commerce platform.

## Quick Start

`@czo/kit` is included automatically as a dependency of all c-zo modules. You don't install it directly.

## Subpath Exports

| Export | Description |
|--------|-------------|
| `@czo/kit` | Logger, core types |
| `@czo/kit/db` | Drizzle ORM, Repository base class, migrations |
| `@czo/kit/graphql` | Schema registration, context factories, codegen helpers |
| `@czo/kit/ioc` | IoC container, `useContainer()` |
| `@czo/kit/event-bus` | Typed event bus, domain events |
| `@czo/kit/nitro` | `defineNitroModule`, plugin helpers |
| `@czo/kit/cache` | Cache utilities |
| `@czo/kit/queue` | Queue abstraction |
| `@czo/kit/telemetry` | Telemetry/observability |
